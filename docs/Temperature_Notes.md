# 2-meter Temperature

Why?

- Standard air temperature at human height.
- Best variable for heatwave prediction.
- Used worldwide in weather forecasting.

Dataset

- Source: IMDAA
- Provider: NCMRWF
- Format: NetCDF (.nc)

Dimensions

- Time
- Height
- Latitude
- Longitude

Units

- Kelvin

Converted to Celsius

°C = K − 273.15
